# Spendify - Expenses Tracker
Spendify is a lightweight expense and budget tracking app that helps you organize spending, manage lists, and stay in control of your finances with ease.

## Tech Stack 
* Vite – Fast and optimized development build tool for modern web apps.
* TanStack – Powerful state and data management for efficient UI updates.
* Dexie.js – IndexedDB wrapper for reliable offline-first data storage.

### Libraries

* Tailwind CSS – Utility-first styling for a clean and responsive UI.
* shadcn/ui – Accessible and reusable UI components with modern design.
* React Hook Form – Efficient form handling with validation and performance optimization.

## Features

* Expense Tracker – Record and view expenses with categorized breakdowns.
* List Maker – Create and manage lists for groceries, tasks, or spending plans.
* Budget Creation & Tracking – Set budgets and track progress to avoid overspending.
